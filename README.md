# AudioTranscriber

AudioTranscriber is a macOS/iOS app that records audio and transcribes it using OpenAI's Whisper API.

## 🚀 Features

- Record audio
- Transcribe using OpenAI Whisper
- Built in Swift
- Simple UI

## 🛠️ Requirements

- Xcode
- Swift 5+
- macOS or iOS
- OpenAI API key
